Machine Learning Models

The following models were implemented to classify the rice grains:

  • Logistic Regression: Accuracy = 90%
  • Decision Tree: Accuracy = 87%
  • Random Forest: Accuracy = 92%
  • SVM: Accuracy = 91%
  • k-NN: Accuracy = 89%
